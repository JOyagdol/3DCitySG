# Graph Schema

Node labels are currently mapped to `CityObject` with a `node_type` property.

Core relation set:

- `CONTAINS`
- `CONSISTS_OF_BUILDING_PART`
- `INTERIOR_ROOM`
- `OUTER_BUILDING_INSTALLATION`
- `INTERIOR_BUILDING_INSTALLATION`
- `ROOM_INSTALLATION`
- `INSIDE`
- `ADJACENT_TO`
- `TOUCHES`
- `INTERSECTS` (planned; extraction not enabled in current pipeline)
- `CONNECTS`
- `BOUNDED_BY`
- `HAS_APPEARANCE`
- `HAS_SURFACE_DATA`
- `APPLIES_TO`
- `HAS_LOD_GEOMETRY`
- `HAS_GEOMETRY_COMPONENT`
